Collection of various SQL Server Scripts.

BlogSQLScripts: This repository contains scripts referred to by the SqlRob.com blog.
ContactInfoScripts: Country/State reference data plus functions to validate contact data.
ImportDara: Text files with data - Mostly for examples for importing data on my blog.
ReferenceData: Scripts to build reference data tables (Currency and Language)
SqlServerFeatures: Scripts to demonstrate new features in various editions of SQL Server.
